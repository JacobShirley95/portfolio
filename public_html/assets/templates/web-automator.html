Java web automator

GitHub

This application is a scriptable web-framework that allows programmatic access to a loaded web page, allowing anyone to read data from the page, communicate via JavaScript, along with many other features. It is powered by jxBrowser.

It has a number of features:

  • A virtual mouse to simulation mouse events in the page.
  • A tab menu to allow multiple pages to be loaded at once.
  • A scriptable API to make finding elements easier.
  • All the features of jxBrowser.
  • Extensible GUI framework.
