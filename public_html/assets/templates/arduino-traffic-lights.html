What it is

As part of my Computer Science course and more specifically a module on Embedded Architecture, I had to simulate a series of junctions using the Arduino Microcomputer and IDE. The code is all written in C and is compiled and uploaded through the IDE to Arduino.

Before the program is run, all the lights and/or buttons need to be connected to the Arduino on the ports specified in each program.
