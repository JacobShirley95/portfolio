Powerbot Script Base

GitHub

This projects provides a foundation on which scripts can be created for Powerbot. It is intended to only include very loose and generic classes which can be used to make a variety of tasks easier.

The base itself uses a State machine to allow code to be separated and allow operations to inherit from each other.
